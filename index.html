毎回の実験で作成したページ

第1回 WWW と HTML

今回はHTMLでの文書作成の方法やその要素について学びました。 演習1-3ではHTMLの基本的な使い方や特別な記号をWebページに表示する方法を学び、 演習1-4では2種類の箇条書きの仕方や画像の掲載方法を知り、ハイパーリンクを文章につけられるようになりました。

演習の成果

  • 演習1-3: HTML文書の作成
  • 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回はCSSを使ったWebページの装飾方法について学びました。 練習問題では、文字のフォントや色を変え方や影の付け方、枠を作り方やグラデージョンの付け方、文章の位置の調整方法といったことを学び、CSSで幅広いことができることを知りました。

演習の成果

  • 練習問題2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

今回はJavaScriptの使用方法について学びました。 JavaScriptを使用することで、たくさんのデータの中から必要なデータを出力することができます。 コンソールに表示させる方法は簡単だったけど、コンソールに複数のものを表示させるときに console.log()とたくさん書かなければいけなくて大変でした。

演習の成果

  • 練習3-6: オブジェクトの参照

今回はJavascriptでのDOMの基礎について学びました。 DOMはJavascriptを用いてHTMLやCSSを書き換えることができるプログラムです。 document.createElementでHTMLに新しい要素を追加したり、.remove()でHTMLのプログラムを消して表示しないようにするなどできることが多いです。 今回ではあまり完璧に使いこなせなかったので、練習して使いこなせるようになりたいです。
