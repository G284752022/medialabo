「Webコンテンツ作成」の成果

本テーマの学習目的

本講義の学習の目的は、HTMLやCSS、JavaScriptの3つの基本的な使用方法を学び理解することです。

そして、ここで教わった内容を使用、応用して簡単なWebページを作成、装飾、イベント機能を付与することが本講義の最終目標です。

プロジェクト課題について

私はプロジェクト課題として、グルメ情報、番組表、世界の天気の中から「世界の天気」を検索して表示するものを選択し、作成するものを選びました。

私は世界の12の都市を文字で検索するのではなくプルダウンで選択できるようにして、 そこで選択した都市の緯度や経度、天気、湿度、温度、風速など9つの項目を表示するようにしました。

プルダウンから都市をもう1度選択すると、前に表示されていた結果が更新されて新しく選択した都市の結果が表示されます。

背景色を水色にすることで画像が一瞬表示されないことがあっても白色の画面にならないように工夫した。

プロジェクト課題で作成したページ: project.html

今回はHTMLでの文書作成の方法やその要素について学びました。 演習1-3ではHTMLの基本的な使い方や特別な記号をWebページに表示する方法を学び、 演習1-4では2種類の箇条書きの仕方や画像の掲載方法を知り、ハイパーリンクを文章につけられるようになりました。

  • 演習1-3: HTML文書の作成

今回はCSSを使ったWebページの装飾方法について学びました。 練習問題では、文字のフォントや色を変え方や影の付け方、枠を作り方やグラデージョンの付け方、文章の位置の調整方法といったことを学び、CSSで幅広いことができることを知りました。

第3回 JavaScript の基本

今回はJavaScriptの使用方法について学びました。 JavaScriptを使用することで、たくさんのデータの中から必要なデータを出力することができます。 コンソールに表示させる方法は簡単だったけど、コンソールに複数のものを表示させるときに console.log()とたくさん書かなければいけなくて大変でした。

演習の成果

  • 練習3-6: オブジェクトの参照

第4回 DOM と JavaScript

今回はJavaScriptでのDOMの基礎について学びました。 DOMはJavaScriptを用いてHTMLやCSSを書き換えることができるプログラムです。 document.createElementでHTMLに新しい要素を追加したり、.remove()でHTMLのプログラムを消して表示しないようにするなどできることが多いです。 今回ではあまり完璧に使いこなせなかったので、練習して使いこなせるようになりたいです。

工夫点・苦心点・強調点・感想

  • [工夫点]: 工夫したところは、Webページを見やすくすために、index.htmlで毎回の講義の内容を記録するところを グリッド状にしてまとめたところです。
    それとあまり派手な色を使わないようにして見づらくならないようにしたところです。
    このホームページを作成するときにモチーフにしたものは季節で、
    ピンク色は春の桜、緑色は夏の葉っぱ、オレンジ色は秋の紅葉、灰色は冬の雪をそれぞれイメージして作成しました。

  • [苦心点]: 特に難しかったところはJavaScriptを使って作業したところです。 JavaScriptで.remove()を使ってHTMLの表示をしないようにすると他のものも表示されないようになったり、 document.querySelector()など長い関数があったりときちんと覚えていないと使えなかったりしたためです。
    それと、CSSで色を変更したかったときに色を変更したくないところまで色が変わってしまって、変更したいものに名前をつけなくてはいけないと言うのが大変でした。

  • [強調点]: このホームページを作成するときに強調した点は、見やすくしたという点です。
    文字が中途半端なところで次の行に改行されてしまう場合に、区切りのいいところであらかじめ改行して見やすくしました。
    さらに、箇条書き状になっているところ数当てゲームやこのレポート部分では間を1行空けて区切りをわかりやすく、見やすくしました。

  • [感想]: 私はこの講義で初めてHTNL、CSS、JavaScriptについて学びました。
    この講義を受講する前は聞いたことがあってホームページに使うということしか知らない状態で、どうやってホームページを作るのかは知りませんでした。
    だから実際にHTML、CSS、JavaScriptに触れる前は、これらを学ぶのはかなり難しいんだろうなと思っていました。
    しかし実際に学んでみるとHTML、CSSの基礎的な部分は思っていたよりも格段に簡単でした。
    JavaScriptに関しては、学んだ箇所もいまだによくわかっていない部分があるので将来的に詳しく学びたいと思っています。
    しかしJavaScriptはJavaのコーディング方法に少し似ている部分があったので、Javaを学んでいる身として、その部分に関してはわかりやすいと思いました。
    ただ、学んできたプログラミングと似ているところもあれば違うところもあるため、同じようにコーディングしてしまうことがあり、
    上手くいかなくなったりしたため、そこで難しいなと再度感じました。
    この講義で学んだことを使って、自分でホームページを作って見たいと思いました。
